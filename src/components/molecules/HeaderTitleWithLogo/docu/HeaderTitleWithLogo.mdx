## Setup Instructions

To use the `HeaderTitleWithLogo` in your project, follow these steps:

1. Import the `HeaderTitleWithLogo` component where you intend to use it:

```jsx
import { HeaderTitleWithLogo } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

```jsx
import { THeaderTitleWithLogo, THeaderTitleWithLogoWithHTMLAttributes } from 'fui-fancyui';
```

## Example Usage

Here's an example demonstrating how to use the `HeaderTitleWithLogo` component within a project:

```jsx
import React from 'react';
import { HeaderTitleWithLogo } from 'fui-fancyui';

const AppHeader = () => (
  <HeaderTitleWithLogo
    title="Your Project Title"
    logo={<img src="path/to/your/logo.png" alt="Logo" />}
    themeType="primary"
    layer="base"
  />
);

export default AppHeader;
```

This example renders a header with a specified logo and title, applying the "primary" theme type and "base" layer settings.

## Component Properties

The `HeaderTitleWithLogo` component accepts the following properties:

- **title** (`ReactNode | string`): The title content to be displayed in the header. This can be a string or any React node.
- **logo** (`ReactNode`): The logo to be displayed alongside the title. This should be a React node, such as an image or an SVG.
- **themeType** (`TUiColorsNotTransparent`): Specifies the color theme type for the heading. Defaults to "secondary" if not provided.
- **layer** (`TLayer`): Determines the layering context for theme color calculation, affecting the text color based on the provided theme.
- **...htmlProps** (`React.HTMLAttributes<HTMLDivElement>`): Any additional standard HTML attributes you wish to apply to the outer wrapper of the component.

This component is designed to be flexible and easily integrated into various header layouts, making it a versatile choice for adding a logo and title to your application's UI.
