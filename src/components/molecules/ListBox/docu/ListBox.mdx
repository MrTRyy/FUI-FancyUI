## Setup Instructions

To use the `ListBox` in your project, follow these steps:

1. Import the `ListBox` component where you intend to use it:

```jsx
import { ListBox } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

```jsx
import { TListBox, TListBoxWithFancyBoxAttrs } from 'fui-fancyui';
```

## Usage

To use `ListBox`, import it into your React component and then use `ListBox.Item` to add list items.

```jsx
import { ListBox } from 'fui-fancyui';

const MyComponent = () => (
  <ListBox>
    <ListBox.Item>Item 1</ListBox.Item>
    <ListBox.Item>Item 2</ListBox.Item>
    // More items...
  </ListBox>
);
```

# ListBoxItem Component

`ListBoxItem` is used within `ListBox` to render individual list items. It is a styled `li` element.

## Usage

`ListBoxItem` is used as a child of `ListBox`.

```jsx
<ListBox.Item>Item 1</ListBox.Item>
```

## Props

`ListBoxItem` accepts the following props:

- `children`: ReactNode - The content of the list item.
- `externalStyle`: CSSProp - Optional external styles to be applied to the list item.
