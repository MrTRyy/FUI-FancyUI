## Setup Instructions

To use the `FancyProgressBar` in your project, follow these steps:

1. Import the `FancyProgressBar` component where you intend to use it:

```jsx
import { FancyProgressBar } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

```jsx
import { TFancyProgressBar, TFancyProgressBarWithHTMLAttrs } from 'fui-fancyui';
```

## Example Usage

Here's a simple example of how to use the `FancyProgressBar` component in a React component:

```jsx
import React from 'react';
import FancyProgressBar from '@/path/to/FancyProgressBar';

const ExampleComponent = () => (
  <FancyProgressBar
    progress={60}
    maxValue={100}
    label="Progress"
    labelAlign="left"
    progressCount={true}
    progressAlign="right"
    themeType="primary"
    textLayer="0"
    layer="0"
  />
);

export default ExampleComponent;
```
