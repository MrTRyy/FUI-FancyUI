## Setup Instructions

To use the `LabeledInput` in your project, follow these steps:

1. Import the `LabeledInput` component where you intend to use it:

```jsx
import { LabeledInput } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

```jsx
import { TLabeledInput } from 'fui-fancyui';
```

### Example Usage

Here's how you can use the `LabeledInput` component within your React application:

```jsx
<LabeledInput
  id="email"
  label="Email Address"
  hasPlaceholder={true}
  inputElement={<input type="email" placeholder="Enter your email" />}
  themeType="primary"
  underline={true}
  isActive={false}
  align="left"
  labelVariant="animated"
/>
```
