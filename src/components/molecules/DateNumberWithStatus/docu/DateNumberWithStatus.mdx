## Setup Instructions

To use the `DateNumberWithStatus` in your project, follow these steps:

1. Import the `DateNumberWithStatus` component where you intend to use it:

```jsx
import { DateNumberWithStatus } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

```jsx
import { TDateNumberWithStatus, TDateNumberWithStatusWithHTMLAttrs } from 'fui-fancyui';
```

## Example Usage

Here's an example of how to use the `DateNumberWithStatus` component within a React component:

```jsx
<DateNumberWithStatus
  dateNumber={15}
  isAvailable={true}
  disabled={false}
  isSelected={true}
  isCurrentDay={false}
  onClick={() => console.log('Date clicked')}
  range={undefined} // Optional: specify if needed
  layer="base"
  themeType="primary"
/>
```

This example demonstrates how to display the date number 15 with an available status, selected state, and using the 'primary' theme type.
