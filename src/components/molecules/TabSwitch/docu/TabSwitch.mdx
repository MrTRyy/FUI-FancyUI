## Setup Instructions

To use the `TabSwitch` in your project, follow these steps:

1. Import the `TabSwitch` component where you intend to use it:

```jsx
import { TabSwitch } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

```jsx
import { TTabSwitch, TTabSwitchWithHTMLAtts } from 'fui-fancyui';
```

### Example Usage

```jsx
<TabSwitch
  values={[
    { itemKey: 'tab1', label: 'Tab 1' },
    { itemKey: 'tab2', label: 'Tab 2' },
  ]}
  textColor="primary"
  sizeC="md"
  wide={false}
  disabled={false}
  tabSpacing="md"
  borderRadius="sm"
  direction="horizontal"
  outlined={true}
  currentSelect="tab1"
  iconAlign="left"
  activeColor="secondary"
  handler={(value) => console.log(`Selected tab: ${value}`)}
/>
```

### List of Used Components

- **FancyTabSwitchButton**: A molecule that represents each selectable tab within the `TabSwitch` component, providing customizable size, color, disabled state, and selection handling.
- **SwitchActiveIndicator**: An atom used to indicate the currently selected tab, customizable in terms of type (blob or underline), color, and orientation.
