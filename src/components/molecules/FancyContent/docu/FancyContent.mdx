## Setup Instructions

To use the `FancyContent` in your project, follow these steps:

1. Import the `FancyContent` component where you intend to use it:

```jsx
import { FancyContent } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

```jsx
import { TFancyContent, TFancyContentHTMLAttrs } from 'fui-fancyui';
```

3. Subcomponents that should used `FancyContent.Icon`, `FancyContent.Title`, and `FancyContent.Description`

import React from 'react';

The `FancyContent` component is designed to be flexible and customizable to fit the needs of various design systems and UI requirements.

### Subcomponents of FancyContent

- **FancyContent.Icon**: Utilizes the `FancySVGAtom` component for rendering icons within the content structure. It's designed for visual elements that complement the textual content.

- **FancyContent.Title**: Employs the `Typography` component to present titles or headings. This subcomponent ensures that titles are consistent in style and appearance across the application.

- **FancyContent.Description**: Makes use of the `Typography` component to display descriptive or supplementary text. It provides a cohesive look and feel for textual content that accompanies the main title or icon.

These subcomponents are integral to the `FancyContent` component, allowing for a modular approach to building rich, informative content blocks with icons, titles, and descriptions. Each subcomponent is tailored to fit its specific role within the content structure, ensuring flexibility and consistency in design.
