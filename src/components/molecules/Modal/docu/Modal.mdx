## Setup Instructions

To use the `Modal` in your project, follow these steps:

1. Import the `Modal` component where you intend to use it:

```jsx
import { Modal } from 'fui-fancyui';
```

2. If using TypeScript, ensure the type definitions are imported for prop validation:

```jsx
import { TModal, TModalWithHTMLAttributes } from 'fui-fancyui';
```

## Example Usage

```jsx
import React, { useState } from 'react';
import { Modal } from 'fui-fancyui';

function App() {
  const [isOpen, setIsOpen] = useState(false);

  return (
    <div>
      <button onClick={() => setIsOpen(true)}>Open Modal</button>
      <Modal isOpen={isOpen} onClose={() => setIsOpen(false)} isCloseable={true} themeType="primary" layer={2}>
        <p>Modal Content Here</p>
      </Modal>
    </div>
  );
}
```
