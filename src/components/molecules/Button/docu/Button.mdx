## Setup Instructions

To use the `Button` in your project, follow these steps:

1. Import the `Button` component where you intend to use it:

```jsx
import { Button } from 'fui-fancyui';
```

2. If using TypeScript, ensure the type definitions are imported for prop validation:

```jsx
import { TButton, TButtonWithNativeAttrs } from 'fui-fancyui';
```

Make sure to adjust the import path according to your project structure.

## Example Usage

Below is an example of how to use the Button component in a React project:

```jsx
<Button
  sizeC="lg"
  wide={true}
  themeType="primary"
  layer={1}
  borderRadius="small"
  outlined={true}
  onClick={() => console.log('Button clicked')}
>
  Click Me
</Button>
```

This example renders a large, full-width button with a primary theme, small border radius, and an outline. It also logs a message to the console when clicked.
