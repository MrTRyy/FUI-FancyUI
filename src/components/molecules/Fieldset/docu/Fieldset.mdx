## Setup Instructions

To use the `Fieldset` in your project, follow these steps:

1. Import the `Fieldset` component where you intend to use it:

```jsx
import { Fieldset } from 'fui-fancyui';
```

2. If using TypeScript, ensure the type definitions are imported for prop validation:

```jsx
import { TFieldset, TFieldsetWithHTMLAttributes } from 'fui-fancyui';
```

## Example Usage

Below is an example demonstrating how to use the `Fieldset` component in a form. This example includes a label with customized typography settings.

```jsx
import React from 'react';
import { Fieldset } from 'fui-fancyui';

function ExampleForm() {
  return (
    <form>
      <Fieldset
        label="Personal Information"
        alignLabel="center"
        typographySettings={{
          elType: 'span',
          variant: 'interactiveLg',
          fontWeight: 'bold',
        }}
        disabled={false}
      >
        {/* Form fields go here */}
      </Fieldset>
    </form>
  );
}
```
