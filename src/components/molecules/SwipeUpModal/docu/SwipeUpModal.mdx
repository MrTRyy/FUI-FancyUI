## Setup Instructions

To use the `SwipeUpModal` in your project, follow these steps:

1. Import the `SwipeUpModal` component where you intend to use it:

```jsx
import { SwipeUpModal } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

```jsx
import { TSwipeUpModal, TSwipeUpModalWithHTMLAttrs } from 'fui-fancyui';
```

## Example Usage

This example demonstrates how to use the `SwipeUpModal` component within a React application:

```jsx
import React, { useState } from 'react';
import { SwipeUpModal } from 'fui-fancyui'; // Adjust the import path as necessary

const App = () => {
  const [isModalOpen, setIsModalOpen] = useState(false);

  const toggleModal = () => setIsModalOpen(!isModalOpen);

  return (
    <div>
      <button onClick={toggleModal}>Toggle Modal</button>
      <SwipeUpModal
        isOpen={isModalOpen}
        isCloseable={true}
        isScalable={true}
        onClose={() => setIsModalOpen(false)}
        themeType="primary"
        layer={1}
        backdrop={true}
      >
        <div>Your content here</div>
      </SwipeUpModal>
    </div>
  );
};

export default App;
```
