## Setup Instructions

To use the `TextAvatar` in your project, follow these steps:

1. Import the `TextAvatar` component where you intend to use it:

```jsx
import { TextAvatar } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

```jsx
import { TTextAvatar, TTTextAvatarWithHTMLAttrs } from 'fui-fancyui';
```

### Example Usage

```jsx
import React from 'react';
import { TextAvatar } from 'fui-fancyui';

const UserProfile = () => {
  return (
    <div>
      <TextAvatar
        text="JD"
        sizeC="lg"
        color="white"
        backgroundColor="blue"
        rounded="medium"
        themeType="dark"
        layer={3}
      />
    </div>
  );
};

export default UserProfile;
```
