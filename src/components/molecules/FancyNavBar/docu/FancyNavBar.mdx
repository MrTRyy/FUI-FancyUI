## Setup Instructions

To use the `FancyNavBar` in your project, follow these steps:

1. Import the `FancyNavBar` component where you intend to use it:

```jsx
import { FancyNavBar } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

```jsx
import { TFancyNavBar, TFancyNavBarWithHTMLProps } from 'fui-fancyui';
```

## Example Usage

Below is an example showcasing how to incorporate the `FancyNavBar` into a project. This example demonstrates a basic navigation bar with customized styling:

```jsx
import React from 'react';
import { FancyNavBar } from 'fui-fancyui';

const MyComponent = () => (
  <FancyNavBar
    externalStyle={css`
      background-color: navy;
      color: white;
    `}
  >
    <a href="/home">Home</a>
    <a href="/about">About</a>
    <a href="/contact">Contact</a>
  </FancyNavBar>
);

export default MyComponent;
```
