## Setup Instructions

To use the `DynamicScrollBar` in your project, follow these steps:

1. Import the `DynamicScrollBar` component where you intend to use it:

```jsx
import { DynamicScrollBar } from 'fui-fancyui';
```

2. If using TypeScript, ensure the type definitions are imported for prop validation:

```jsx
import { TDynamicScrollBar } from 'fui-fancyui';
```

## Example Usage

Below is an example demonstrating how to use the `DynamicScrollBar` component in a React project:

```jsx
import React from 'react';
import { DynamicScrollBar } from 'fui-fancyui';

const MyComponent = () => (
  <DynamicScrollBar scrollable={true} activateScrollbar={true}>
    <div>Your content here</div>
  </DynamicScrollBar>
);

export default MyComponent;
```

This example wraps the content within a `DynamicScrollBar`. The scrollbar will be activated and visible only if both `scrollable` and `activateScrollbar` props are set to `true`.
