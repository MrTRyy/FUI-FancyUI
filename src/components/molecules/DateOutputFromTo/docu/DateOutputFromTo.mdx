## Setup Instructions

To use the `DateOutputFromTo` in your project, follow these steps:

1. Import the `DateOutputFromTo` component where you intend to use it:

```jsx
import { DateOutputFromTo } from 'fui-fancyui';
```

2. If using TypeScript, ensure the type definitions are imported for prop validation:

```jsx
import { TDateOutputFromTo, TDateOutputFromToWithNativeAttrs } from 'fui-fancyui';
```

## Example Usage

Here's a basic example of how to incorporate the `DateOutputFromTo` component into a React component:

```jsx
import React from 'react';
import { DateOutputFromTo } from 'fui-fancyui';

function ExampleComponent() {
  const handleDateChange = (which) => {
    console.log(`${which} date selected`);
  };

  return (
    <DateOutputFromTo
      dateFrom={new Date(2023, 0, 1)}
      dateTo={new Date(2023, 11, 31)}
      handleFromTo={handleDateChange}
      themeType="primary"
      sizeC="md"
    />
  );
}
```
