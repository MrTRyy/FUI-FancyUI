## Setup Instructions

To use the `SingleInput` component in your project, follow these steps:

1. Import the `SingleInput` component where you intend to use it:

```jsx
import { SingleInput } from 'fui-fancyui';
```

2. If using TypeScript, ensure the type definitions are imported for prop validation:

```jsx
import { TSingleInputAtom, TSingleInputAtomWithHTMLAttrs } from 'fui-fancyui';
```

## Example Usage

Here is an example of how to use the `SingleInput` component within a form meant for a verification process:

```jsx
import React, { useRef } from 'react';
import { SingleInput } from 'fui-fancyui';

function VerificationForm() {
  const inputRef = useRef(null);

  const handleKeyDown = (e) => {
    console.log('Key pressed:', e.key);
  };

  return (
    <SingleInput
      ref={inputRef}
      value=""
      ariaLabel="Verification Input"
      onKeyDown={handleKeyDown}
      themeType="primary"
      layer="base"
    />
  );
}
```
