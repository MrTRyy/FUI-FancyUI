## Setup Instructions

1. Ensure you have React and the necessary project setup to support React components.
2. Import the `PasswordEye` component into your project file where you intend to use it:

```jsx
import { PasswordEye } from 'fui-fancyui';
```

3. Make sure to import the type definitions if you're using TypeScript for prop type validation:

```jsx
import { TPasswordEye, TPasswordEyeSVGProps } from 'fui-fancyui';
```

## Example Usage

Here's a basic example of how to integrate the `PasswordEye` component within a form to toggle password visibility:

```jsx
import React, { useState } from 'react';
import { PasswordEye } from 'fui-fancyui';

function PasswordInput() {
  const [isShow, setIsShow] = useState(false);

  const toggleVisibility = () => setIsShow(!isShow);

  return (
    <div>
      <input type={isShow ? 'text' : 'password'} />
      <PasswordEye isShow={isShow} onClick={toggleVisibility} />
    </div>
  );
}
```
