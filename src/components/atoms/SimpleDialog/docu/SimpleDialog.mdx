## Example Usage

Here's an example of how to use the `SimpleDialog` component to create a modal dialog that can be opened or closed based on application state:

```jsx
import React, { useState } from 'react';
import {SimpleDialog} from '@tobitry/fui-fancyui';

const App = () => {
  const [isOpen, setOpen] = useState(false);

  return (
    <>
      <button onClick={() => setOpen(true)}>Open Dialog</button>
      <SimpleDialog isOpen={isOpen} themeType="primary" layer={1} onClose={() => setOpen(false)}>
        <p>Dialog content goes here. You can place any React component inside.</p>
      </SimpleDialog>
    </>
  );
};

export default App;
````

This example demonstrates opening a `SimpleDialog` with a button click, showcasing how to integrate the dialog into a React application seamlessly.
