# SystemMessage Component

## Overview

The `SystemMessage` component displays text messages with colors indicating different system states like success, error, info, or warning.

## Usage

```jsx
import SystemMessage from '@/components/atoms/SystemMessage';

<SystemMessage systemMessageState="error">Error message</SystemMessage>;
```

## Props

- **systemMessageState**: Defines the message type (`'success'`, `'error'`, `'info'`, `'warning'`).
- **children**: The message text.
- **textSettings** _(optional)_: Additional styling for the text.

## Implementation

This component utilizes styled-components for dynamic styling based on the `systemMessageState`.
