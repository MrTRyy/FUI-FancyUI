## Setup Instructions

To use the `SwipeUpContainer` in your project, follow these steps:

1. Import the `SpacingContainer` component where you intend to use it:

```jsx
import { SwipeUpContainer } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

```jsx
import { TSwipeUpContainer, TSwipeUpContainerWithHTMLAttrs } from 'fui-fancyui';
```

## Example Usage

Here's a simple example of how to use the `SwipeUpContainer` component:

```jsx
<SwipeUpContainer isScalable={true} themeType="secondary" layer={1} style={{ height: '50%' }}>
  <p>Your content goes here</p>
</SwipeUpContainer>
```

This example creates a `SwipeUpContainer` that is scalable, uses the "secondary" theme type, is on layer 1, and has a height of 50%.
