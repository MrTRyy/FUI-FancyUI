## Setup Instructions

To incorporate the `FancyTextOverlay` component into your project, follow these steps:

1. Import the `FancyTextOverlay` component into your component file:

   ```jsx
   import { FancyTextOverlay } from 'fui-fancyui';
   ```

2. Import necessary types if you intend to explicitly type your component props:

   ```jsx
   import { TFancyTextOverlay, TImageVideoOverlayWithHTMLAttrs, TTextOverlayPositions } from 'fui-fancyui';
   ```

## Example Usage

Below is an example showing how to use the `FancyTextOverlay` component to overlay text on a video or image element:

```jsx
import React from 'react';
import { FancyTextOverlay } from 'fui-fancyui';
import YourVideoOrImageComponent from './YourVideoOrImageComponent';

function ExampleWithTextOverlay() {
  return (
    <FancyTextOverlay position="bottom-right" textChildren={<p>Your overlay text here</p>}>
      <YourVideoOrImageComponent />
    </FancyTextOverlay>
  );
}
```

This example demonstrates wrapping a video or image component with `FancyTextOverlay`, specifying the position of the text overlay and including the desired text.
