## Setup Instructions

To use the `FancySpacingBox` component in your project, follow these steps:

1. Import the `FancySpacingBox` component where you intend to use it:

```jsx
import { FancySpacingBox } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

```jsx
import { TSpacingsContainerWithHTMLAttrs, TSpacingsContainer } from 'fui-fancyui';
```

3. Use the component in your JSX, passing in the appropriate props for padding, margin, and any other HTML or styling properties you wish to apply.

## Example Usage

```jsx
<FancySpacingBox padding={['20px', '30px']} margin="10px" as="section">
  <p>Your content here</p>
</FancySpacingBox>
```

This example renders a `FancySpacingBox` as a `<section>` element, with 20px padding on the top and bottom, 30px padding on the left and right, and a margin of 10px around it. Replace `"Your content here"` with your own content or components.
