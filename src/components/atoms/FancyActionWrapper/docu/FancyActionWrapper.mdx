# FancyActionWrapper Component

The `FancyActionWrapper` is a React component designed to wrap its children in either a styled `<a>` tag or a `<button>` or Custome element, depending on the specified properties. It can also utilize a custom wrapper component if provided.
I built it to be used in a NextJs application, where the `Link` component is used to wrap anchor tags.

## Input/Output

### Input

- **children** (`React.ReactNode`): The content to be wrapped within the `<a>`, `<button>` tag or custom element.
- **WrapperComponent** (`React.ReactElement`, optional): A custom component to be used as the wrapper instead of the default `<a>` or `<button>` like NextLink.
- **TNativeButtonProps** (`React.HTMLAttributes<HTMLButtonElement> & { as?: 'button' }`): Props specific to the `<button>` element, used when `as` is set to `'button'`.
- **TNativeAnchorProps** (`React.AnchorHTMLAttributes<HTMLAnchorElement> & { as: 'a' }`): Props specific to the `<a>` element, used when `as` is set to `'a'`.

### Output

- **Return Value**: A React component that renders either a styled `<a>` tag, a `<button>`, or a custom `WrapperComponent` containing the `children`.

## Example Usage

```jsx
import React from 'react';
import FancyActionWrapper from 'path-to-FancyActionWrapper';

const ExampleComponent = () => (
  <FancyActionWrapper as="a" href="https://example.com">
    Click me!
  </FancyActionWrapper>
);

export default ExampleComponent;


const ExampleComponent = () => (
  <FancyActionWrapper WrapperComponent={<Link href="http://fui.cool"/>}>
    Click me!
  </FancyActionWrapper>
);

export default ExampleComponent;
```

In this example, `FancyActionWrapper` is used to create a styled anchor (`<a>`) element with the text "Click me!". The `href` attribute makes it function like a regular hyperlink.
