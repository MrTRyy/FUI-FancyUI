## Usage

Integrating the `DisabledBox` is straightforward:

```jsx
import { DisabledBox } from 'fui-fancyui'; // Assuming it's in the same directory

function MyComponent() {
  const isDisabled = true;

  return (
    <DisabledBox disabled={isDisabled}>
      {/* Your content that you want to disable */}
      <button>This button will be disabled</button>
    </DisabledBox>
  );
}
```
