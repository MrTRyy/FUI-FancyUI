## Setup Instructions

To use the `FancyImageWrapper` component in your project, follow these steps:

1. Ensure you have `react` and `styled-components` installed in your project.
2. Import the `FancyImageWrapper` component into your file:

```jsx
import { FancyImageWrapper } from 'fui-fancyui';

// the type definitions are also available for TypeScript users
import { TFancyImageWrapper, TFancyImageWrapperNativeAttrs, TFancyImageWrapperWithNativeAttrs } from 'fui-fancyui';
```

3. Import necessary types if you are using TypeScript for prop definitions:

```jsx
import { TFancyImageWrapperWithNativeAttrs, TFancyImageWrapper, TFancyImageWrapperNativeAttrs } from 'fui-fancyui';
```

## Example Usage

Below is an example demonstrating how to use the `FancyImageWrapper` component to wrap an image with custom styling:

```jsx
<FancyImageWrapper
  aspectRatio="16/9"
  borderRadius="10px"
  sizeW="300px"
  sizeH="fit"
  filter={{ blur: '2px', brightness: 0.9 }}
  objectFit="cover"
>
  <img src="path/to/image.jpg" alt="Sample Image" />
</FancyImageWrapper>
```
