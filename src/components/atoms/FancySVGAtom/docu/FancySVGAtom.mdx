## Setup Instructions

To use the `FancySVGAtom` component in your project, follow these steps:

1. Import the `FancySVGAtom` component into your file:

   ```jsx
   import { FancySVGAtom } from 'fui-fancyui';
   ```

2. Make sure to import the required types and utilities if you're planning to use them for the component props:
   ```jsx
   import { TFancySVGAtomSizes, TFancySVGAtom, TFancySVGAtomWithNativeAttrs } from 'fui-fancyui';
   ```

## Example Usage

Here's an example demonstrating how to use the `FancySVGAtom` component within a React application:

```jsx
import React from 'react';
import { FancySVGAtom } from 'fui-fancyui';
import { ReactComponent as YourIconSVG } from './path/to/your-icon.svg';

function ExampleComponent() {
  return (
    <FancySVGAtom sizeC="md" isActive themeType="primary">
      <YourIconSVG />
    </FancySVGAtom>
  );
}
```

In this example, the `FancySVGAtom` wraps a custom SVG component (`YourIconSVG`), applying medium size, active state, and primary theme color.
