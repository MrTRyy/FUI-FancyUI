## Setup Instructions

To use the `FancyColorInput` in your project, follow these steps:

1. Import the `FancyColorInput` component where you intend to use it:

```jsx
import { FancyColorInput } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:


## Example Usage

The example below demonstrates how to integrate the `FancyColorInput` component into a React application, including the setup for initial color, color type, and handlers for various interactions:

```jsx
<FancyColorInput
  pickedColor={'hsla(210, 100%, 50%, 0.5)'}
  opacity={0.5}
  currentColorType="hsla"
  handler={(color) => {
    console.log('Color changed:', color);
  }}
  colorTypeHandler={(type) => {
    console.log('Color format changed to:', type);
  }}
  handlerOpacity={(opacity) => {
    console.log('Opacity changed to:', opacity);
  }}
/>
```

This setup provides a user interface for changing color values and formats dynamically, with changes reflected in real-time and communicated back to the parent component.
