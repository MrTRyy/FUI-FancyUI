## Setup Instructions

To use the `FancyTextArea` component:

1. **Import the Component:**

   ```jsx
   import FancyTextArea from 'fui-fancyui';
   ```

   (or wherever your component is located)

2. **(Optional) Import Type Definitions (TypeScript):**
   ```typescript
   import { TFancyTextArea } from 'fui-fancyui';
   ```

## Example Usage

```jsx
<FancyTextArea
  id="description-input"
  label="Description"
  placeholder="Enter a detailed description..."
  value={descriptionText}
  onChange={handleDescriptionChange}
  themeType="secondary"
  layer={3}
  systemMessage={validationError ? 'error' : ''}
  icon={<InfoIcon />}
/>
```
