## Setup Instructions

To use the `FancyPasswordInput` in your project, follow these steps:

1. Import the `FancyPasswordInput` component where you intend to use it:

```jsx
import { FancyPasswordInput } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:


## Example Usage

Here's an example demonstrating how to use the `FancyPasswordInput` component in a form:

```jsx
import React from 'react';
import { FancyPasswordInput } from 'fui-fancyui';

function LoginForm() {
  return (
    <form>
      <FancyPasswordInput
        placeholder="Enter your password"
        label="Password"
        icon={<SVG />} // Assume you have an icon system setup
        systemMessage={{
          type: 'error',
          message: 'This is an error message',
        }}
      />
      {/* Other form elements */}
    </form>
  );
}
```

## List of Used Components

- **PasswordInput**: A basic password input component allowing users to enter passwords. This component is designed to handle password visibility toggling and validation.
- **InputWrapper**: A higher-order component that wraps an input with additional UI elements like labels, icons, and system messages. It's used to enhance the form input experience by providing consistent styling and behavior.
