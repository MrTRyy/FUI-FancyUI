## Setup Instructions

To use the `FancyCodeVerificationInput` in your project, follow these steps:

1. Import the `FancyCodeVerificationInput` component where you intend to use it:

```jsx
import { FancyCodeVerificationInput } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

```jsx
import { TFancySingleInputs, TFancySingleInputsWithHTMLAttrs } from 'fui-fancyui';
```

### Example Usage

```jsx
import React from 'react';
import { FancyCodeVerificationInput } from 'fui-fancyui';

function App() {
  const handleVerification = (code) => {
    // API call or validation logic here
    console.log(code);
  };

  return (
    <FancyCodeVerificationInput
      length={6}
      errorMessage="Invalid code"
      isSuccess={false}
      handler={handleVerification}
      automaticCase="upper"
      debounceTime={500}
    />
  );
}
```
