## Setup Instructions

To use the `FancyChipList` in your project, follow these steps:

1. Import the `FancyChipList` component where you intend to use it:

```jsx
import { FancyChipList } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

```jsx
import { TFancyChipList, TChip } from 'fui-fancyui';
```

### Example Usage

```jsx
import React from 'react';
import  { FancyChipList, TFancyChipList, TChip } from 'fui-fancyui';

const initialChips: TChip[] = [{ id: '1', label: 'React' }, { id: '2', label: 'Styled-Components' }];

function App() {
  return (
    <FancyChipList
      chips={initialChips}
      themeType="primary"
      inputPlaceholder="Type and press enter..."
      editable={true}
      label="Your Skills"
    />
  );
}
```

### List of Used Components

- **Fieldset**: A component that wraps the `FancyChipList` with a label and applies typography settings. Used for grouping elements and providing a label.
- **ChipList**: Displays a list of chips with customizable theme, layer, and outlined properties. It's a foundational component for managing a collection of chips.
- **FancyChip**: A template component that represents each chip in the list. It is customizable, interactive, and can be focused or editable.
