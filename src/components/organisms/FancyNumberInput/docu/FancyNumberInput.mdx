## Setup Instructions

To use the `FancyNumberInput` in your project, follow these steps:

1. Import the `FancyNumberInput` component where you intend to use it:

```jsx
import { FancyNumberInput } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

```jsx
import { TFancyNumberInput } from 'fui-fancyui';
```

After installing the required library, you can import and use the `FancyNumberInput` component in your project as follows:

### Example Usage

```jsx
<FancyNumberInput
  value={10}
  label="Age"
  icon={<AgeIcon />}
  disabled={false}
  systemMessage={{ type: 'error', message: 'This is an error message' }}
  align="left"
  id="age-input"
  themeType="dark"
  layer="primary"
  autoWidth={true}
  placeholder="Enter your age"
  onFocus={() => console.log('Focused')}
  onBlur={() => console.log('Blurred')}
/>
```
