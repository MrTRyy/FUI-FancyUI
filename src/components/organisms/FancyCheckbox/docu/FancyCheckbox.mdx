## Setup Instructions

To use the `FancyCheckbox` in your project, follow these steps:

1. Import the `FancyCheckbox` component where you intend to use it:

```jsx
import { FancyCheckbox } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

```jsx
import { TFancyCheckbox } from 'fui-fancyui';
```

### Example Usage

```jsx
<FancyCheckbox
  label="Agree to Terms"
  description="Click to agree to the terms and conditions"
  defaultChecked={false}
  onChange={(e) => console.log(e.target.checked)}
  align="center"
  alignCheckbox="left"
/>
```
