## Setup Instructions

To use the `FancyListBox` in your project, follow these steps:

1. Import the `FancyListBox` component where you intend to use it:

```jsx
import { FancyListBox } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

```jsx
import { TFancyListBox, TFancyListBoxWithListBoxAttrs } from 'fui-fancyui';
```

After installing the required library, you can import and use the `FancyListBox` component in your project as follows:

### Example Usage

```jsx
import React from 'react';
import { FancyListBox } from 'fui-fancyui';

function App() {
  return (
    <FancyListBox
      themeType="primary"
      layer={2}
      boxTitle={{ title: { ...TypographyProps }, titleAlign: 'center' }}
      seperator={{ color: 'grey', thickness: '2px' }}
      itemSettings={{ size: 'md' }}
    >
      {yourDataArray.map((dataItem, index) => (
        <div key={index}>{dataItem}</div>
      ))}
    </FancyListBox>
  );
}
```

### List of Used Components

- **FancyLine**: A simple line component used for separators between list items. Allows customization of color, thickness, and other visual aspects.
- **ListBox**: A container for list items. Supports theming, custom sizes, and layers.
- **TitleForComponent**: A component for rendering an optional title above the list, with customizable alignment and styling.
- **ListBox.Item**: Represents an individual item within the `ListBox`, which can be customized with padding, margin, and other settings.

This documentation covers the setup and usage of the `FancyListBox` component, detailing its properties and the components it utilizes for enhanced functionality and visual presentation.
