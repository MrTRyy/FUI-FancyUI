## Setup Instructions

To use the `FancySearchBar` in your project, follow these steps:

1. Import the `FancySearchBar` component where you intend to use it:

```jsx
import { FancySearchBar } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

```jsx
import { TFancySearchBar, TFancySearchBarWithHTMLProps } from 'fui-fancyui';
```

After installing the required library, you can import and use the `FancySearchBar` component in your project as follows:

### Example Usage

```jsx
import { FancySearchBar, TFancySearchBarWithHTMLProps } from 'fui-fancyui';

function App() {
  const handleSearchValue = (value: string) => {
    console.log(value);
  };

  return (
    <FancySearchBar
      handlerSearchValue={handleSearchValue}
      value="Initial Search Value"
      themeType="primary"
      sizeC="sm"
      openListWhenFocused={true}
    />
  );
}
```
