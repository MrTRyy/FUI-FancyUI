#### List of Used Components

- **SystemMessageBox**: A basic container for system messages, accommodating various themes and sizes.
- **FancyContent**: A flexible layout component that manages the orientation and spacing of content and icons within the system message.
- **Icons (SVGStopSign, SVGInfoSign, SVGWarningSign, SVGCircleCheck)**: Various SVG icons used to represent different message types based on the theme.
