## Setup Instructions

To use the `FancyGridTemplateItem` in your project, follow these steps:

1. Import the `FancyGridTemplateItem` component where you intend to use it:

```jsx
import { FancyGridTemplateItem } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

