## Setup Instructions

To use the `FancySkeletonGrid` in your project, follow these steps:

1. Import the `FancySkeletonGrid` component where you intend to use it:

```jsx
import { FancySkeletonGrid } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

```jsx
import { TFancySkeletonGrid } from 'fui-fancyui';
```

After installing the required library, you can import and use the `FancySkeletonGrid` component in your project as follows:

### Example Usage

```jsx
<FancySkeletonGrid
  gridAreas={['header', 'sidebar', 'content']}
  gridAreasBreakPoints={{ mobile: ['header', 'content'], desktop: ['header', 'sidebar', 'content'] }}
  layer="base"
  themeType="light"
  borderRadius="4px"
/>
```

### List of Used Components

- **SkeletonBox**: A component that renders a placeholder box, mimicking the size and shape of the content that will eventually load in its place.
- **FancyGridTemplate**: Provides a flexible and responsive grid layout system. This component is used to arrange the skeleton boxes according to the defined grid areas and breakpoints.
