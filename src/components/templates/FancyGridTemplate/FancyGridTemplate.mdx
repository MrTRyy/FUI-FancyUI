# FancyGridTemplate Component

The `FancyGridTemplate` component is designed to create a flexible and customizable grid layout. It utilizes styled-components and can be used to structure content in a grid format with specified areas, gaps, and dimensions.

## Props

### `FancyGridTemplate` Props

- `gridAreas`: Array of strings defining the grid layout. Each string represents a row in the grid, and each space-separated value within the string defines a grid area.
- `gapColumn`: (Optional) The gap size between columns. Defaults to `'sm'`.
- `gapRow`: (Optional) The gap size between rows. Defaults to `'sm'`.
- `height`: The height of the grid container.
- `width`: (Optional) The width of the grid container. Defaults to `'100%'`.
- `children`: React nodes to be rendered within the grid.

### Usage

```jsx
<FancyGridTemplate
  gridAreas={['header header', 'sidebar content', 'footer footer']}
  gridAreasBreakPoints={{
    sm: ['header', 'sidebar', 'content', 'footer'],
    md: ['header header', 'sidebar content', 'footer footer'],
  }}
  gapColumn="md"
  gapRow="md"
  height="auto"
  width="80%"
>
  {/* children components here */}
</FancyGridTemplate>
```

## Styling

The `StyledFancyGridTemplate` is a styled `div` element that applies the grid layout based on the provided props. It sets the width, height, row and column gaps, and grid template areas.

## GridItem Sub-component

`FancyGridTemplate.GridItem` is a sub-component used to position children within the grid.

### `FancyGridTemplateItem` Props

- `gridArea`: The grid area where the component should be placed.
- `externalStyle`: (Optional) Additional styled-components syntax for custom styling.
- `children`: React nodes to be rendered within the grid item.

### Usage

```jsx
<FancyGridTemplate.GridItem gridArea="header">{/* children components here */}</FancyGridTemplate.GridItem>
```

## Styling

The `GridItem` is a styled `div` that receives the `grid-area` and any additional external styling.

## Export

The `FancyGridTemplate` and its sub-component `FancyGridTemplateItem` are exported for use in other components.

## Example

```jsx
import React from 'react';
import FancyGridTemplate, { FancyGridTemplateItem } from './path-to-your-component';

const MyGridPage = () => {
  return (
    <FancyGridTemplate
      gridAreas={['header header', 'sidebar content', 'footer footer']}
      gapColumn="md"
      gapRow="md"
      height="100vh"
    >
      {/* Header */}
      <FancyGridTemplate.GridItem gridArea="header">
        <div style={{ backgroundColor: '#ffcc00' }}>Header</div>
      </FancyGridTemplate.GridItem>

      {/* Sidebar */}
      <FancyGridTemplate.GridItem gridArea="sidebar">
        <div style={{ backgroundColor: '#00ccff' }}>Sidebar</div>
      </FancyGridTemplate.GridItem>

      {/* Content */}
      <FancyGridTemplate.GridItem gridArea="content">
        <div style={{ backgroundColor: '#cccccc' }}>Content</div>
      </FancyGridTemplate.GridItem>

      {/* Footer */}
      <FancyGridTemplate.GridItem gridArea="footer">
        <div style={{ backgroundColor: '#ff6699' }}>Footer</div>
      </FancyGridTemplate.GridItem>
    </FancyGridTemplate>
  );
};

export default MyGridPage;
```
