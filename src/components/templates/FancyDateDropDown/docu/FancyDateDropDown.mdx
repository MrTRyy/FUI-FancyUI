## Setup Instructions

1. Import the `FancyChip` component where you intend to use it:

```jsx
import { FancyChip } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

```jsx
import { TFancyDateDropDown, TFancyDateDropDownWithFancyDropDownSelect, TDateType } from 'fui-fancyui';
```

### Example Usage

```jsx
import { FancyDateDropDown } from 'fui-fancyui';

const MyComponent = () => {
  return <FancyDateDropDown type="month" dateLimits={[1, 12]} placeholder="Select Month" />;
};
```
