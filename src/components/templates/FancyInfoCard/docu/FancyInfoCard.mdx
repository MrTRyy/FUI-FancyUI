## Setup Instructions

To use the `FancyInfoCard` in your project, follow these steps:

1. Import the `FancyInfoCard` component where you intend to use it:

```jsx
import { FancyInfoCard } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

```jsx
import { TFancyInfoCardProps, TFancyInfoCardWithFancyBoxAndFlex } from 'fui-fancyui';
```

After installing the required library, you can import and use the `FancyInfoCard` component in your project as follows:

### Example Usage

```jsx
<FancyInfoCard
  title="Your Title Here"
  description="Your description here."
  icon={<YourIconComponent />}
  sizeC="lg"
  direction="row"
  align="center"
  justify="start"
/>
```
