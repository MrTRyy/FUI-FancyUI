## Setup Instructions

1. Import the `FancyFlexBox` component where you intend to use it:

```jsx
import { FancyFlexBox } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

```jsx
import { TFancyFlexBox, TFancyFlexBoxWithDynamicElAttrs } from 'fui-fancyui';
```

### Example Usage

```jsx
<FancyFlexBox align="center" direction="row" justify="space-between" gap="10px" separator={<span>|</span>} fitBox>
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</FancyFlexBox>
```
