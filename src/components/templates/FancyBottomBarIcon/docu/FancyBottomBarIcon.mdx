## Setup Instructions

1. Import the `FancyBottomBarIcon` component where you intend to use it:

```jsx
import { FancyBottomBarIcon } from 'fui-fancyui';
```

2. If using TypeScript, you can import the type definitions for prop validation:

```jsx
import { TFancyBottomBarIcon } from 'fui-fancyui';
```

### Example Usage

```jsx
<FancyBottomBarIcon
  WrapperComponent={YourCustomWrapper}
  type="button"
  href="#"
  onClick={() => console.log('Icon clicked')}
  iconProps={
    {
      /* Icon properties here */
    }
  }
/>
```

<Canvas />

- `YourCustomWrapper` refers to the component you want to use as a wrapper for the icon.
- `iconProps` should include any props specific to the `BottomBarIcon` component, such as the icon's name or style properties.
