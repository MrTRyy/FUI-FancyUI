# ComponentAndActionWrapper

`ComponentAndActionWrapper` is a versatile React component designed to handle different actions with various elements. It's particularly useful in scenarios like creating a Navbar, where you might need to use different types of elements like `<a>`, `<button>`, or Next.js's `<Link>` for navigation.

This component allows for flexible rendering of children within different types of wrappers, enhancing reusability and consistency throughout the application.

## Props

This component accepts the following props:

- `WrapperComponent` (optional): A React component or element to be used as the wrapper for the children.
- `children` (optional): The content or elements to be wrapped.
- `type` (optional): Specifies the type of action element. Can be either `'button'` or `'a'` for anchor.
- `href` (optional): The URL to be used for an anchor tag. This is mandatory when `type` is `'a'`.
- `onClick` (optional): A function to be called when the button or anchor is clicked.

## Usage

### With a Custom Wrapper Component

```jsx
import ComponentAndActionWrapper from './path/to/ComponentAndActionWrapper';
import MyCustomWrapper from './path/to/MyCustomWrapper';

<ComponentAndActionWrapper WrapperComponent={<MyCustomWrapper />}>
  <p>Content to be wrapped</p>
</ComponentAndActionWrapper>
```

### As a Button

```jsx
<ComponentAndActionWrapper type="button" onClick={() => console.log('Button clicked')}>
  Click Me
</ComponentAndActionWrapper>
```

### As an Anchor

```jsx
<ComponentAndActionWrapper type="a" href="https://example.com">
  Go to Example.com
</ComponentAndActionWrapper>
```

## Example in a Navbar

This component is ideal for use in a Navbar where various types of elements are needed. Below is an example showing how `ComponentAndActionWrapper` can be used with Next.js's `<Link>` component along with standard HTML elements.

```jsx
import React from 'react';
import Link from 'next/link';
import ComponentAndActionWrapper from './ComponentAndActionWrapper';

function Navbar() {
  return (
    <nav>
      <ComponentAndActionWrapper type="a" href="/home">
        Home
      </ComponentAndActionWrapper>

      <ComponentAndActionWrapper type="button" onClick={() => console.log('About Us clicked')}>
        About Us
      </ComponentAndActionWrapper>

      <ComponentAndActionWrapper WrapperComponent={<Link href="/contact" passHref><a>Contact</a></Link>}>
        <p>Contact Page</p>
      </ComponentAndActionWrapper>
    </nav>
  );
}

export default Navbar;
```

In this Navbar example, we're using `ComponentAndActionWrapper` to handle an anchor tag, a button, and a Next.js `Link` component. This showcases the component's flexibility in handling various UI elements, making it a valuable asset for building interactive and responsive web applications.
