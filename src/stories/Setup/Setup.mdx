## Step 2: Integrate FancyThemeProvider ðŸŽ¨

Integrate the `FancyThemeProvider` in your application to manage the theme using the provided `dummyTheme` variable:

```javascript
import { FancyThemeProvider } from '@mrtryy/fancyuilibary';
import { createGlobalStyle } from 'styled-components';

// Define your Global Style
const GlobalStyle = createGlobalStyle`
  body {
    margin: 0;
    padding: 0;
    background: ${({ theme }) => theme.colors.primary[0]};
    color: ${({ theme }) => theme.colors.secondary[0]};
    font-family: DMSans, Helvetica, Sans-Serif;
  }
  // Add additional global styles as needed
`;

const dummyTheme = {
  // Theme configuration goes here
  // ...
};

export default function App({ children }) {
  return (
    <FancyThemeProvider theme={dummyTheme}>
      <GlobalStyle />
      {children}
    </FancyThemeProvider>
  );
}
```

In this setup, the `FancyThemeProvider` is directly using the `dummyTheme` variable as its theme, ensuring a consistent and appealing UI design across your application.
