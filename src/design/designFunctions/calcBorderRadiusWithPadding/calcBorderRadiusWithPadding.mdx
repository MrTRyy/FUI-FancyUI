## Component Title

`calcBorderRadiusWithPadding`

## Brief Description

This component is a utility function designed to calculate and return a new CSS border-radius value by adding or subtracting padding values from the original border-radius. It supports theme values (`'sm'`, `'md'`, `'lg'`) and explicit string values (e.g., `'12px'`, `'24'`) for both border-radius and padding. This function is flexible, accepting single values or arrays of values for both parameters, and includes an option to specify whether the padding should be inset (subtracted) or outset (added).

## Example Usage

To use `calcBorderRadiusWithPadding` in your project, call the function with your desired border-radius and padding values. Here is an example that demonstrates both inset and outset padding calculations:

```javascript
// For inset padding
const insetBorderRadius = calcBorderRadiusWithPadding(['md', 'lg'], ['12px', '24px'], true);
console.log(insetBorderRadius); // Example output: "calc(16px - 12px) calc(24px - 24px)"

// For outset padding
const outsetBorderRadius = calcBorderRadiusWithPadding('sm', '12px', false);
```

## Component Properties

- **borderRadius** (`TThemeArrayOrValueCSS`): The original border-radius values. Can be a single value or an array of values. Accepts theme values (`'sm'`, `'md'`, `'lg'`) or explicit string values (e.g., `'12px'`).
- **padding** (`TThemeArrayOrValueCSS`): The padding values to add or subtract from the border-radius. Can be a single value or an array of values, similar to borderRadius.
- **inset** (`boolean`): Optional. Specifies whether the padding should be inset (subtracted) or outset (added). Defaults to `true`.
