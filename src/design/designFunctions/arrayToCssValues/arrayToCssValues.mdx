# `arrayToCssValues` Function Documentation

The `arrayToCssValues` function is an advanced utility designed for React applications, enabling developers to dynamically calculate CSS values for properties such as padding, margin, and border-radius. It allows for a more flexible and intuitive way of defining these values using either single values or arrays that represent different edges of an element. This function is particularly useful in theme-driven designs where consistency and ease of maintenance are key.

## Brief Description

This function extends the flexibility of specifying CSS values within your application by allowing inputs as either single values or arrays of up to four values, corresponding to the top, right, bottom, and left edges of an element, respectively. It integrates seamlessly with a theme store, making it easier to apply consistent spacing, border-radius, or other CSS properties across the application based on predefined theme values.

## Setup Instructions

1. **Import the Function**: Include the `arrayToCssValues` function in your component or utility file by importing it as shown:

```jsx
import { arrayToCssValues } from 'fui-fancyui';
```

## Example Usage

```jsx
// Assuming you're using styled-components for styling
import styled from 'styled-components';
import { arrayToCssValues } from 'fui-fancyui';

const StyledBox = styled.div`
  padding: ${() => arrayToCssValues(['md', 'lg'], 'spacing')};
  padding: ${() => arrayToCssValues('12px', 'spacing')};
  border-radius: ${() => arrayToCssValues(['sm', 'md', 'lg', 'xl'], 'borderRadius')};
`;

// Usage in a React component
const MyStyledComponent = () => (
  <StyledBox>This box has variable padding and border-radius based on theme values.</StyledBox>
);
```

## Component Properties

- **values** (`TThemeArrayOrValueCSS`): This can be a single value or an array of values. When an array is provided, each value corresponds to a side of the element starting from the top and moving clockwise. It supports both theme-defined values and direct CSS values.

- **themeSetting** (`keyof TValueMap | undefined`): Specifies the theme property to apply, such as `borderRadius` or `spacing`. This is used to fetch the correct values from the theme store.

## Functionality Details

- **Single or Multiple Values**: The function can process both single values and arrays of values, adapting the CSS output accordingly. Single values apply uniformly to all sides, while arrays can specify different values for each side.

- **Theme Integration**: When a theme setting is specified, the function looks up the corresponding value in the theme store. This facilitates the use of predefined theme values for consistent styling across the application.

- **Flexible Outputs**: Based on the input array's length, the function generates CSS strings that match common CSS shorthand rules, e.g., one value for all sides, two values for vertical/horizontal sides, etc.

## Notes

- The function gracefully handles undefined values within the input array, ensuring robustness in various usage scenarios.
- It's essential to have a well-defined theme structure and ensure that the theme store is properly integrated within your application for this function to work effectively.
- The versatility of `arrayToCssValues` simplifies the application of responsive design principles, allowing for easy adjustments to spacing and border-radius based on screen size or other factors.
